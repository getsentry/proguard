# Changelog

## 4.0.0

This is a complete rewrite of the crate. It focuses on the `Mapping` type, which
provides high level metadata about a proguard mapping file, and allows iterating
over the contained `Record`s, and the `Mapper`, which allows re-mapping class
names and entire frames, with support for inlined frames as well.

## 3.0.0

- Update `uuid` to `0.8.1`.

## 2.0.0

- Implement support for R8.
- Update `uuid` to `0.7.2`.

## 1.1.0

- Update `uuid` to `0.6.0`.

## 1.0.0

- Initial stable release.
